jQuery hash Demo - Github

Demo for how to use jQuery hash plugin

Write "ref=search" to Hash
Code: $.hash("ref","search");

Write "abc=123" to Hash
Code: $.hash("abc",123);

Read the value of "ref"
Code: $.hash("ref");

Change the value of "ref" from "search" to "affliate"
Code: $.hash("ref","affliate");

Delete the property "abc" frm Hash
Code: $.hash("abc",null);

Clear everything in hash using: window.location.hash='';
